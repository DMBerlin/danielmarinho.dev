## ğŸ’» Running locally

1. Clone this repo:

```sh
$ git clone https://github.com/DMBerlin/danielmarinho.dev
```

2. Then go to the project's folder:

```sh
cd danielmarinho.dev
```

3. Install all dependencies:

```sh
pnpm install
```

4. Run locally:

```sh
pnpm dev -o
```
